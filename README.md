# Instructions to Run the App

1. **Clone the repository:**
    ```sh
    git clone https://github.com/yourusername/stackai.git
    cd stackai
    ```

2. **Install dependencies:**
    ```sh
    pnpm install
    ```

3. **Start the application:**
    ```sh
    pnpm run dev
    ```

4. **Open your browser and navigate to:**
    ```
    http://localhost:3000
    ```
